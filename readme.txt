A powerful plugin to customize WordPress admin, frontend, and login page settings with features like custom CSS, maintenance mode, and more.

== Description ==

**T9 Admin Settings** provides advanced tools to enhance and customize your WordPress site. Manage admin options, set maintenance mode, customize login pages, and override CSS for both admin and frontend. This plugin is designed to help you create a tailored WordPress experience.

**Features Include:**
- Add custom favicon across admin and frontend.
- Modify admin footer text with dynamic placeholders like `[CURRENT_YEAR]`, `[WEBSITE_TITLE]`, and more.
- Remove WordPress version numbers for better security.
- Hide the admin toolbar on the frontend.
- Enable/disable automatic WordPress updates.
- Customize the login page with custom logo, background, and CSS.
- Apply custom CSS for admin and frontend.
- Maintenance mode with custom messages and countdown timers.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/t9admin/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "T9 Settings" in the WordPress admin menu to configure the plugin.

== Frequently Asked Questions ==

= How do I enable Maintenance Mode? =
Navigate to "T9 Settings" > "Maintenance", enable Maintenance Mode, and set a custom message or end date if desired.

= Can I add custom CSS to the admin or frontend? =
Yes, go to "T9 Settings" > "Custom Code" and add your CSS for admin or frontend.

= Does this plugin support custom login page design? =
Yes, you can customize the login page with a custom logo, background image, and CSS from the "Login" tab in the settings.
